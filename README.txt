This Python Script is used to scrape information from the web by firing simple http calls.
It retrieves the tree hierarchical information by rcursive fetches of HTML documents.
Nature of iinformation: standard occupations values (international standard SOC).
Purpose of this data is to serve as seed data for a tool which allows users to easily capture, search and transmit 
occupation information of parties to the middleware.

This is an end user program and does not have utility feature. 
However, the source has been shared because of the solution approach and tools used to solve simple problems.
This is not an attempt to generalization  of similar or related problems. In that sense, its just a piece of software for reference, be it for me or anyone else who bumps into a similar or related problem.
